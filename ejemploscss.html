Pagina de ejempplos de CSS

Este es un ejemplo de cómo se pueden usar los comandos en CSS para dar estilo a una página web.

  • Un comando para cambiar el color de fondo de una página:

    • body { background-color: #f0f0f0; }

  • Un comando para cambiar el color de texto:

    • body { color: #333; }

  • Un comando para cambiar el tamaño de fuente:

    • body { font-size: 16px; }

  • Un comando para cambiar el tipo de fuente:

    • body { font-family: Arial, sans-serif; }

  • Un comando para cambiar el margen:

    • body { margin: 20px; }

    o

    body { margin-top: 10px; margin-right: 20px; margin-bottom: 30px; margin-left: 40px; }

  • Un comando para cambiar el borde:

    • body { border: 1px solid #333; }

    o

    body { border-top: 2px dashed red; border-right: 1px solid #333; border-bottom: 3px dotted blue; border-left: 1px double green; }

  • Un comando para cambiar el espacio entre líneas:

    • body { line-height: 1.5; }

    o

    body { line-height: 1.2em; }
